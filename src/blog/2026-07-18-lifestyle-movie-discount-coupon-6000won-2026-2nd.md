## 2차 물량 205만 장, 7월 8일부터 배포 중

정부 영화 관람료 할인권 2차 물량 **205만 장**이 2026년 7월 8일 오전 10시부터 배포되기 시작했습니다. 1차 225만 장에 이어 이번 2차까지 합치면 **총 450만 장** 규모입니다.

오늘(7월 18일)은 2차 배포가 시작된 지 열흘이 지난 시점입니다. 별도의 마감일이 공지되지 않아도, **선착순 예산 소진 방식**인 만큼 언제 끝날지 알 수 없습니다.

## 신청 필요 없음 — 로그인만 하면 자동 지급

가장 헷갈리는 부분이 "따로 신청해야 하나?"인데, 그럴 필요가 없습니다.

- **대상**: CGV·롯데시네마·메가박스·씨네Q 회원
- **수령 방법**: 각 극장 홈페이지 또는 앱에 로그인 후 **마이페이지 '쿠폰함'** 확인
- **신청·다운로드**: 불필요 — 회원이면 자동 지급

## ⚠️ 쿠폰함에 있어도 소멸될 수 있는 이유

여기가 이 글에서 가장 중요한 부분입니다. 할인권은 정가에서 6,000원을 차감해주는 방식인데, **쿠폰을 보유하고 있는 것과 실제로 할인이 적용되는 것은 별개**입니다.

<div class="callout-warning">⚠️ 주의: 할인 지원금은 정해진 예산 안에서 '실제 결제 완료' 순서로 차감됩니다. 예산이 먼저 소진되면, 쿠폰함에 쿠폰이 남아 있어도 결제 시 할인이 적용되지 않고 소멸될 수 있습니다. "쿠폰함에 있으니 나중에 써도 되겠지"라는 생각이 가장 위험합니다.</div>

즉 쿠폰함에 표시되는 건 "받을 자격"일 뿐, "확보된 할인"이 아닙니다. 실제로 예매 결제를 마쳐야 할인이 확정됩니다.

## 사용 조건 체크리스트

예매 전에 아래 조건을 미리 확인해두면 결제 단계에서 당황하지 않습니다.

- [ ] CGV·롯데시네마·메가박스·씨네Q 중 회원으로 가입된 극장 앱/홈페이지 로그인
- [ ] 마이페이지 '쿠폰함'에서 할인권 보유 여부 확인
- [ ] **온라인(홈페이지·앱) 예매**로 진행 — 현장 창구 적용 여부는 다를 수 있으니 앱 우선
- [ ] 할인 적용 후에도 **최소 1,000원은 본인 결제**해야 발급 완료됨을 인지
- [ ] 1인 **최대 2매**까지 사용 가능
- [ ] 조조·문화가 있는 날 등 다른 할인과 중복 여부는 **결제 화면에서 직접 확인**

## 극장 4사, 쿠폰 확인 위치가 조금씩 다릅니다

| 극장 | 쿠폰 확인 위치 | 지급 방식 |
|------|--------------|-----------|
| CGV | 앱/홈페이지 로그인 → MY → 쿠폰함 | 회원이면 자동 지급 |
| 롯데시네마 | 앱/홈페이지 로그인 → 마이페이지 → 쿠폰·할인권 | 회원이면 자동 지급 |
| 메가박스 | 이벤트 페이지에서 '할인권 받기' → 마이페이지 | '받기'를 눌러야 담김 |
| 씨네Q | 이벤트 페이지 또는 쿠폰함에서 확인 | 이벤트에서 확인 권장 |

CGV·롯데시네마는 회원이면 로그인만 해도 쿠폰함에 들어와 있지만, 메가박스처럼 이벤트 페이지에서 '할인권 받기'를 눌러야 담기는 곳도 있습니다. 그냥 쿠폰함만 봤다가 "왜 안 들어와 있지?" 하기 쉬우니, 본인이 이용할 극장의 방식부터 확인하세요.

## 정리

- 2차 할인권 205만 장, 7월 8일 오전 10시부터 배포(1차 225만 장 + 2차 205만 장 = 총 450만 장)
- CGV·롯데시네마·메가박스·씨네Q 회원이면 **로그인만 해도** 쿠폰함에 자동 지급
- 할인 후 **최소 1,000원 본인 결제** 필요, 1인 **최대 2매**
- **쿠폰함에 있어도 예산 소진 시 자동 소멸** — 보이는 즉시 예매·결제까지 끝내는 게 안전
- 다른 할인과의 중복 여부는 결제 화면에서 직접 확인

쿠폰함에 할인권이 떠 있다면, 오늘 볼 영화가 없어도 지금 예매 화면까지 들어가 확인해보세요. "나중에 써야지"가 "이미 소진됐어요"가 되는 건 순식간입니다.
